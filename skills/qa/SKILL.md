# Blacksmith QA Skill

## Trigger
테스트, 회귀, Android 기기 검증, 밸런스 관찰 또는 출시 준비에 사용한다.

## Minimum checks
- JSON 구문과 참조 무결성
- 제작 진행도·피버 경계값
- 정밀작업 ON/OFF
- 강화 이정표 수식어
- 고객 방문 결과 인센티브
- 상인 납품 가격 가산
- 저장·복귀·방치 보상
- 세로 화면, 노치, 태블릿·폴더블 비율

## Status language
PASS, PARTIAL, FAIL, NOT_RUN을 사용하고 실행하지 않은 검증을 PASS로 표시하지 않는다.
