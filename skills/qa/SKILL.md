# Blacksmith QA

## Modes

- `static`
- `model`
- `runtime`
- `device`
- `regression`
- `evidence`

## Minimum automated checks

- JSON 구문·ID 참조
- 제작 진행·피버·정밀 마감
- 일반/특수 강화 분리
- 성장·가격 가속
- 촉매 가격·효과
- +11 하락·+30 파괴
- 안정 단조 보호
- 폭주 2단계 도약과 금지 위치
- +10 수식어·+100 4티어
- 보관함·자동 단조 경계
- 문서·Registry·경로 Governance

## Manual/device checks

- 실제 화면·스크롤·텍스트
- 정밀 입력
- 장시간 연타 피로
- 노치·안전 영역
- 휴대전화·태블릿·폴더블
- Android AAB
- 접근성·성능

## Status

`PASS / PARTIAL / FAIL / NOT_RUN`. 테스트 파일에 적힌 case 수와 실제 실행 로그가 일치해야 한다.
