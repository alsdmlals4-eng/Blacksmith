# Blacksmith Game Design Skill

## Trigger
제작, 강화, 수식어, 재료 조합, 고객, 경제, MVP 범위를 변경할 때 사용한다.

## Read first
- `[기획서]/01_통합_게임_기획/BLACKSMITH_GAME_BIBLE.md`
- 관련 `data/crafting/` 또는 `data/sales/`

## Rules
- 간단한 모바일 광클 경험을 최우선으로 둔다.
- 직원과 수리 시스템을 임의로 추가하지 않는다.
- 설명 규칙과 실제 JSON 값을 중복 소유하지 않는다.
- 변경 후 관찰 가능한 플레이 결과를 정의한다.

## Validation
- 고객 방문과 상인 납품의 역할이 겹치지 않는가?
- 재료가 강화 수식어와 숨은 조합에 실제 영향을 주는가?
- 직접 플레이와 방치 플레이가 모두 진행 가능한가?
