# 특수 강화·위험·자동 단조 검증

## 분류

- 일반 강화: +10 단위가 아닌 모든 단계, 원클릭 판정
- 특수 강화: +10·+20·…·+100
- 특수 강화에서만 보조재료·촉매·정밀 판정 적용
- 일반 강화에 이전 특수 강화 효과가 누출되지 않음

## 실패·위험

- +1~+10 실패: 유지
- +11부터: 단계 하락 가능
- +30부터: 파괴 가능
- 실패당 성공률 +4%p, 최대 +24%p
- 성공 시 실패 보정 초기화

## 단조 기술

- 균형: 기본
- 안정: 파괴 0, 하락 위험 감소, 비용 증가
- 폭주: 성공 시 8% 확률로 총 2단계 상승
- 폭주는 특수 강화와 도약 범위에 특수 강화가 포함되는 위치에서 사용 불가

## 수식어

- +10 첫 수식어
- +30 두 번째 수식어
- +50 세 번째 수식어
- +100 세 수식어 모두 4티어

## 보관함·자동 단조 수동 경계

- 목표 도달 후 자동 보관
- 반복 시 6칸에서 종료
- 골드 부족 시 종료
- 지정 재료가 없으면 빈 슬롯으로 계속
- 파괴 시 반복 설정에 따라 새 무기 시작 또는 종료
- 폭주 도약이 목표 단계를 넘길 때 균형 단조로 대체

자동 모델 테스트와 실제 UI·Android 검증은 별도 상태로 기록한다.
