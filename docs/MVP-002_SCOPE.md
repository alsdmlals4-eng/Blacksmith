# MVP-002 +100 강화·위험·보관·자동 단조 POC

## 목표

빠른 일반 강화와 10단계마다 선택을 요구하는 특수 강화의 리듬, 고단계 성장·가격·손실 위험, 안전/도약 선택, 자동 반복 생산이 모바일 POC로 성립하는지 검증한다.

## 플레이 흐름

```text
철검
→ +1~+9 일반 강화
→ +10 특수 강화
→ 같은 10단계 리듬 반복
→ 원하는 단계에서 보관 또는 +100
→ 자동 단조로 목표 단계 반복 생산
```

## 강화 분류

### 일반 강화

- +10 단위가 아닌 모든 목표 단계
- 버튼 한 번으로 성공·유지·하락·파괴 판정
- 보조재료·촉매·정밀 판정 미사용

### 특수 강화

- +10·+20·…·+100
- 보조재료·촉매 선택
- 정밀 판정
- 수식어 추가·성장
- 일반 강화보다 높은 비용·성장·위험

## 성장·가격

- 고정 공격력 덧셈이 아니라 현재 강화 공격력에 대한 비율 성장
- 일반 기본 성장률 8%, 10단계 구간마다 +1%p
- 특수 기본 성장률 20%, 10단계 구간마다 +2%p
- 강화 비용과 판매가는 단계·공격력에 따라 가속
- 다음 공격력·판매가·비용·확률을 시도 전에 표시

실제 수치 정본: `data/crafting/enhancement_balance.json`

## 성공률·실패 보정

10단계 기본 패턴: 98% / 95% / 92% / 89% / 86% / 83% / 80% / 77% / 74% / 68%.

- 완료된 10단계 구간마다 -2.5%p
- 누적 감소 최대 -25%p
- 최저 기본 성공률 25%
- 실패당 +4%p, 최대 +24%p
- 성공 시 실패 보정 초기화

## 위험

- +1~+10: 실패 시 유지
- +11부터: 단계 하락 가능
- +30부터: 파괴 가능
- 고단계일수록 하락 비중·파괴 비중·하락 단계 증가
- 파괴 시 세션 종료, 공격력·판매가 0, 보관 불가
- 하락 후에도 실패 보정 유지

## 단조 기술

- 균형 단조: 기본
- 안정 단조: 비용 1.8배, 성장 0.95배, 파괴 0, 하락 위험 0.5배
- 폭주 단조: 성공 시 8% 확률로 총 2단계 상승
- 폭주는 특수 강화와 도약 범위에 특수 강화가 포함되는 위치에서 사용 불가
- 자동 단조 목표를 초과할 수 있으면 균형 단조로 대체

## 재료·촉매

- 보조재료: 숫돌, 화염석, 정령의 심장
- 촉매: 살라맨더의 핵, 수호 가루, 폭주의 불씨
- 효과·가격 정본: `data/crafting/materials.json`
- 특수 강화에서만 적용
- 자동 단조에서 지정 재고가 없으면 빈 슬롯으로 계속

## 수식어 성장

- +10 첫 수식어 1티어
- +20 첫 수식어 2티어
- +30 두 번째 수식어 1티어
- +40 두 번째 수식어 2티어
- +50 세 번째 수식어 1티어
- +60 세 번째 수식어 2티어
- +70·+80·+90 각 수식어 3티어
- +100 모든 수식어 4티어

## 보관함·자동 단조

- POC 보관함 6칸
- 목표 강화 단계 지정
- 목표 도달 시 자동 보관
- 보관함이 찰 때까지 반복
- 골드·보조재료·촉매 재고 소비
- 골드 부족·수동 중지·보관함 가득 참 시 종료
- 파괴 시 반복 설정에 따라 새 철검으로 재시작
- 현재 자원·보관함은 메모리 전용
