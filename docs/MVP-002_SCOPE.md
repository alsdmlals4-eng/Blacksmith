# MVP-002 강화 이정표 조작 테스트

## 목표

철검을 +20까지 강화하면서 일반 단계는 원클릭으로 빠르게 진행하고, 수식어가 변화하는 +5·+10·+15·+20에서만 정밀 강화가 등장하는 조작 주기를 검증한다.

## 플레이 흐름

`철검 완성 → 보조재료·촉매 선택 → 4단계 원클릭 강화 → 이정표 정밀 강화 → 수식어 변화 → 다음 구간 반복 → +20 완료`

## 강화 조작 주기

- +1~+4: 원클릭 즉시 판정
- +5: 정밀 강화, 첫 수식어 1티어 추가
- +6~+9: 원클릭 즉시 판정
- +10: 정밀 강화, 첫 수식어 2티어 강화
- +11~+14: 원클릭 즉시 판정
- +15: 정밀 강화, 두 번째 수식어 1티어 추가
- +16~+19: 원클릭 즉시 판정
- +20: 정밀 강화, 두 번째 수식어 2티어 강화

일반 단계에는 정밀 강화 화면이 나타나지 않는다. 강화 버튼을 누르는 즉시 성공·실패가 판정된다.

## 포함

- 철검 +0에서 +20까지 단계별 강화
- 숫돌·화염석·정령의 심장 보조재료 선택
- 촉매 없음·살라맨더의 핵 선택
- 단계별 기본 성공률
- 촉매와 연속 실패 보정이 반영된 최종 성공률
- 이정표에서만 GOOD·PERFECT 정밀 강화 보너스 적용
- 실패 시 재료 소모, 단계 유지, 파괴·수리 없음
- 실패할수록 다음 시도 성공률이 증가하는 누적 보정
- 각 5단계 구간의 보조재료 성질 누적
- +5·+15에서 해당 구간 성질에 맞는 수식어 추가
- +10·+20에서 기존 수식어 티어 강화
- UI와 분리된 강화 상태 모델 및 자동 테스트
- 제작 없이 강화만 실행하는 Godot 테스트 Scene
