# MVP-002 +5 강화와 첫 수식어

## 목표

MVP-001에서 완성한 철검을 +5까지 강화하고, 강화에 사용한 보조재료의 누적 성질에 따라 첫 수식어가 붙는 경험을 검증한다.

## 플레이 흐름

`철검 완성 → 강화 진입 → 보조재료·촉매 선택 → 강화 확률 확인 → 선택적 정밀 강화 → 성공·실패 → +5 도달 → 첫 수식어 획득`

## 포함

- 철검 +0에서 +5까지 단계별 강화
- 숫돌·화염석·정령의 심장 보조재료 선택
- 촉매 없음·살라맨더의 핵 선택
- 단계별 기본 성공률
- 촉매·정밀작업·연속 실패 보정이 반영된 최종 성공률 표시
- 정밀 강화 ON/OFF
- 실패 시 재료 소모, 단계 유지, 파괴·수리 없음
- 실패할수록 다음 시도 성공률이 증가하는 누적 보정
- 강화 시 사용한 재료 성질 누적
- +5 도달 시 가장 강한 재료 성질에 맞는 1티어 수식어 획득
- 강화 결과 요약과 새 철검 제작 복귀
- UI와 분리된 강화 상태 모델 및 자동 테스트

## 초기 수치

- +1: 100%
- +2: 90%
- +3: 80%
- +4: 70%
- +5: 60%
- 살라맨더의 핵: +15%p
- 정밀 강화 GOOD: +10%p
- 정밀 강화 PERFECT: +20%p
- 연속 실패 보정: 실패당 +5%p, 최대 +20%p

## 수식어 후보

- 숫돌 누적 우세: `날카로운`
- 화염석 누적 우세: `불타는`
- 정령의 심장 누적 우세: `정령이 깃든`

동률이면 가장 최근 강화 시도에서 사용한 보조재료의 성질을 우선한다.

## 제외

- +10 수식어 강화
- +15 두 번째 수식어
- 숨은 무기 변환
- 재료 보유량·구매·소비 경제
- 강화 피버 입력
- 고객·상인 판매
- 저장·복귀
- 실제 Android export와 실기기 검증

## 완료 기준

- 정밀작업 OFF에서도 +5까지 진행할 수 있다.
- 촉매와 정밀 판정에 따라 표시 성공률과 실제 판정이 일치한다.
- 실패해도 철검이 파괴되거나 강화 단계가 내려가지 않는다.
- 실패 보정이 다음 시도 성공률에 적용되고 성공 시 초기화된다.
- +5에서 누적 재료 성질에 맞는 첫 수식어가 붙는다.
- Godot 4.7.1 헤드리스 파싱, 강화 모델 테스트, JSON 검증이 통과한다.
