# MVP-002 일반 강화·특수 강화 테스트

## 목표

철검을 +100까지 강화하면서 빠른 원클릭 일반 강화와, 10단계마다 재료 선택과 정밀 판정이 결합된 특수 강화의 반복 리듬을 검증한다.

## 플레이 흐름

`철검 완성 → +1~+9 일반 강화 → +10 특수 강화 → 다음 10단계 구간 반복 → +100 완료`

## 강화 분류

### 일반 강화

- 대상: +10 단위가 아닌 모든 단계
- 조작: 버튼 한 번으로 즉시 성공·실패 판정
- 보조재료: 사용하지 않음
- 촉매: 사용하지 않음
- 정밀 판정: 사용하지 않음

### 특수 강화

- 대상: +10·+20·+30·…·+100
- 보조재료 선택
- 촉매 선택
- 정밀 판정
- 수식어 추가 또는 티어 성장

일반 강화에서는 이전 특수 강화에서 선택한 보조재료와 촉매 효과가 적용되지 않는다. 정밀 판정 보너스도 특수 강화에만 적용한다.

## 포함

- 철검 +0에서 +100까지 단계별 강화
- 일반 강화 원클릭 판정
- 특수 강화 전용 보조재료·촉매 UI
- 특수 강화 전용 정밀 판정
- 숫돌·화염석·정령의 심장 보조재료
- 촉매 없음·살라맨더의 핵
- 10단계 반복 성공률 패턴
- 높은 강화 구간의 점진적 성공률 감소
- 촉매와 실패 보정이 반영된 최종 성공률
- 실패 시 단계 유지, 파괴·수리 없음
- 수식어 3개 추가·티어 성장
- UI와 분리된 강화 상태 모델 및 자동 테스트
- 제작 없이 강화만 실행하는 Godot 테스트 Scene

## 성공률

10단계 안에서 같은 성공률 패턴을 반복한다.

| 구간 위치 | 기본 확률 | 조작 |
|---:|---:|---|
| 1 | 98% | 일반 강화 |
| 2 | 95% | 일반 강화 |
| 3 | 92% | 일반 강화 |
| 4 | 89% | 일반 강화 |
| 5 | 76% | 일반 강화 |
| 6 | 88% | 일반 강화 |
| 7 | 85% | 일반 강화 |
| 8 | 82% | 일반 강화 |
| 9 | 79% | 일반 강화 |
| 10 | 68% | 특수 강화 |

- 10강 구간이 올라갈 때마다 기본 성공률 -2%p
- 누적 감소 최대 -18%p
- 최저 기본 성공률 35%
- 살라맨더의 핵: 특수 강화 성공률 +15%p
- 특수 강화 GOOD: +10%p
- 특수 강화 PERFECT: +20%p
- 연속 실패 보정: 실패당 +5%p, 최대 +20%p

## +100 수식어 성장

- +10: 첫 수식어 1티어 추가
- +20: 첫 수식어 2티어
- +30: 두 번째 수식어 1티어 추가
- +40: 두 번째 수식어 2티어
- +50: 세 번째 수식어 1티어 추가
- +60: 세 번째 수식어 2티어
- +70: 첫 수식어 3티어
- +80: 두 번째 수식어 3티어
- +90: 세 번째 수식어 3티어
- +100: 모든 수식어 4티어

## 수식어 후보

- 숫돌: `날카로운`
- 화염석: `불타는`
- 정령의 심장: `정령이 깃든`
