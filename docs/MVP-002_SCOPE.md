# MVP-002 +100 강화·보관·자동 단조

## 목표

철검을 +100까지 강화하면서 빠른 일반 강화, +10 단위 특수 강화, 고단계 위험·보상, 무기 보관과 목표 기반 자동 단조가 하나의 반복 가능한 Prototype 루프로 동작하는지 검증한다.

## 플레이 흐름

```text
철검 완성
→ +1~+9 일반 강화
→ +10 특수 강화
→ 고단계 하락·파괴·실패 보정
→ +100 수식어 완성
→ 수동 또는 자동 보관
→ 새 철검 / 자동 반복
```

## 강화 분류

### 일반 강화

- +10 단위가 아닌 모든 단계
- 버튼 한 번으로 즉시 판정
- 보조재료·촉매·정밀 판정 미사용
- 이전 특수 강화 선택 효과 비누출

### 특수 강화

- +10·+20·…·+100
- 보조재료·촉매 선택
- 정밀 판정
- 수식어 추가 또는 티어 성장
- 자동 단조에서는 지정 재료 재고가 있을 때만 소비하고, 없으면 해당 재료 없이 진행

## 점진 성장·가치

- 제작의 품질 적용 기본 공격력이 강화 성장의 시작점이다.
- 제작 가치 배율은 강화 후 판매가 계산에 유지된다.
- 공격력 증가는 고정 덧셈이 아니라 현재 강화 적용 공격력과 목표 구간을 기준으로 계산한다.
- 일반 강화보다 특수 강화의 성장률이 크다.
- 고단계일수록 다음 공격력 증가량, 판매가, 강화 비용이 가속된다.
- 화면에 현재 공격력·판매가와 다음 성공 시 공격력·판매가·비용·성공/유지/하락/파괴 확률을 표시한다.
- 실제 값은 `data/crafting/enhancement_balance.json`이 책임진다.

## 실패·위험

- +1~+10: 실패 시 기본적으로 단계 유지
- +11부터: 단계 하락 가능
- +30부터: 무기 파괴 가능
- 높은 10단계 구간일수록 하락·파괴 위험과 하락 단계가 증가
- 실패 보정은 실패 횟수에 따라 다음 성공률을 올리고 성공 시 초기화
- 파괴된 무기는 보관할 수 없고 해당 강화 세션이 종료됨

## 단조 방식

### 균형 단조

기본 성공률·비용·성장·위험을 사용한다.

### 안정 단조

비용이 증가하지만 파괴를 막고 단계 하락 위험을 낮춘다.

### 폭주 단조

- 성공 시 낮은 확률로 총 2단계 상승
- 성장률·파괴 위험을 직접 증폭하는 이전 구조는 사용하지 않음
- 특수 강화에서는 사용 불가
- 2단계 도약으로 +10 이정표를 건너뛸 수 있는 +9·+19·+29·… 구간에서 사용 불가
- 도약이 발생하지 않으면 1단계 상승

도약 확률과 단조 방식 값은 `enhancement_balance.json`이 책임진다.

## 보조재료·촉매

- 보조재료: 숫돌·화염석·정령의 심장
- 촉매: 없음·살라맨더의 핵·수호 가루·폭주의 불씨
- 가격·성공률·성장·판매 가치·하락·파괴 효과를 특수 강화 화면에 표시
- 실제 재료 정의와 가격은 `data/crafting/materials.json`이 책임진다.
- 일반 강화에는 선택 UI와 효과가 적용되지 않는다.

## 수식어 성장

- +10: 첫 수식어 1티어
- +20: 첫 수식어 2티어
- +30: 두 번째 수식어 1티어
- +40: 두 번째 수식어 2티어
- +50: 세 번째 수식어 1티어
- +60: 세 번째 수식어 2티어
- +70: 첫 수식어 3티어
- +80: 두 번째 수식어 3티어
- +90: 세 번째 수식어 3티어
- +100: 모든 수식어 4티어

수식어 후보:

- 숫돌 → `날카로운`
- 화염석 → `불타는`
- 정령의 심장 → `정령이 깃든`

## 무기 보관함

- 최대 6개
- 강화 도중 원하는 단계에서 수동 보관 가능
- 목표 단계 도달 시 자동 단조가 자동 보관
- 이름·강화 단계·원본/품질 적용/강화/최종 공격력·제작 가치 배율·판매가·누적 비용·손익·수식어·촉매·마감 품질 표시
- 파괴 무기는 보관 불가


## 공유 경제 거래

- 수동 강화와 자동 단조는 동일 `WorkshopResources` 인스턴스의 골드·재료 재고를 사용한다.
- 강화 시작 전에 시도 비용과 선택 재료 재고를 검증한다.
- 골드가 부족하면 강화 판정을 시작하지 않는다.
- 수동 특수 강화에서 선택한 보조재료·촉매가 부족하면 골드와 재료를 소비하지 않고 차단한다.
- 자동 단조는 지정 재료가 없으면 해당 슬롯을 비운 뒤 같은 거래 경로로 진행한다.
- 결제 후 강화 시작이 실패하면 차감한 골드와 재료를 복구한다.
- 세션의 `total_spent`와 실제 보유 골드 차감은 같은 시도 비용을 사용한다.

## 자동 단조

설정:

- 목표 강화 단계
- 보관함이 찰 때까지 반복 여부
- 단조 방식
- 특수 강화 보조재료
- 특수 강화 촉매

실행 규칙:

1. 현재 무기를 목표 단계까지 빠르게 강화한다.
2. 특수 강화 목표에서 지정 재료 재고를 확인한다.
3. 재고가 있으면 소비하고, 없으면 해당 슬롯을 비운 상태로 계속한다.
4. 목표 도달 시 무기를 보관한다.
5. 반복 설정이면 새 철검으로 다시 시작한다.
6. 골드 부족·보관함 가득 참·수동 중지에서 종료한다.
7. 파괴 시 반복 설정이면 새 철검으로 재시작하고 아니면 종료한다.
8. 유한 반복과 중지 조건을 보장한다.
