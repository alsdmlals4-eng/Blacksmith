# Godot 제작·강화·자동 단조 테스트 실행

## 필요한 버전

- Godot 4.7.1 stable
- 프로젝트 브랜치: `main`

## 사용자가 할 일

1. GitHub Desktop에서 Blacksmith 저장소를 선택한다.
2. `Fetch origin` 후 `Pull origin`을 누른다.
3. Godot에서 저장소 루트의 `project.godot`을 연다.
4. F5를 누른다.

현재 `project.godot`의 기본 실행 Scene은 강화 테스트 Scene이다.

## 제작 품질 확인

- `scenes/main/main.tscn`을 열고 F6로 전체 제작 흐름을 실행한다.
- 화면 버전이 `POC v0.6.4 · main · 2026.07.23.1`인지 확인한다.
- 보통 마감: 원본 공격력 20 → 품질 적용 20, 제작 가치 ×1.00
- 좋은 마감: 원본 공격력 20 → 품질 적용 21, 제작 가치 ×1.05
- 완벽한 마감: 원본 공격력 20 → 품질 적용 22, 제작 가치 ×1.12
- 피버 1회 이상이면 공격력 ×1.05·제작 가치 ×1.03이 한 번만 더해지는가
- 여러 번 발동해도 결과 보너스가 중첩되지 않는가
- 완벽 마감+피버가 공격력 23·제작 가치 ×1.15인가
- 강화 화면과 보관함에서 원본·마감·피버·합산 제작 배율이 유지되는가
- 반복 자동 단조의 두 번째 무기부터는 보통 마감·피버 미적용인가

## 강화 분류

### 일반 강화

- +10 단위가 아닌 모든 단계
- 버튼 한 번으로 즉시 성공·실패 판정
- 보조재료·촉매·정밀 판정 미사용
- 이전 특수 강화 선택 효과 비누출

### 특수 강화

- +10·+20·…·+100
- 보조재료·촉매 선택
- 정밀 판정
- 수식어 추가 또는 티어 성장

## 위험·성장 확인

- 공격력과 판매가가 현재 강화 적용 기준으로 점진 증가하는가
- 고단계일수록 다음 효과·판매가·비용 증가가 커지는가
- 성공·유지·하락·파괴 확률이 표시되는가
- +11부터 단계 하락이 발생할 수 있는가
- +30부터 무기 파괴가 발생할 수 있는가
- 실패 보정이 누적되고 성공 시 초기화되는가
- 안정 단조와 보호 촉매가 표시된 위험을 실제로 줄이는가

## 폭주 단조 확인

- 성공 시 낮은 확률로 총 2단계 상승하는가
- 도약이 없으면 1단계 상승하는가
- +10·+20·… 특수 강화에서 선택할 수 없는가
- +9·+19·+29·…에서 선택할 수 없는가
- 어떤 경우에도 특수 강화 수식어 이정표를 건너뛰지 않는가

## 수식어 성장

- +10: 첫 수식어 1티어
- +20: 첫 수식어 2티어
- +30: 두 번째 수식어 1티어
- +40: 두 번째 수식어 2티어
- +50: 세 번째 수식어 1티어
- +60: 세 번째 수식어 2티어
- +70·+80·+90: 각 수식어 3티어
- +100: 모든 수식어 4티어

## 보관함 확인

- 강화 도중 `강화 종료 및 보관`이 가능한가
- 최대 6개까지만 보관되는가
- 이름·강화 단계·원본/제작 적용/강화/최종 공격력·마감·피버·합산 제작 가치·판매가·비용·손익·수식어·촉매·마감 품질이 표시되는가
- 파괴 무기는 보관되지 않는가

## 수동·자동 경제 확인

- 일반 강화를 한 번 누르면 표시된 시도 비용만큼 보유 골드가 감소하는가
- 특수 강화에서 선택한 보조재료와 촉매가 시도 시작 시 각각 1개 감소하는가
- 골드가 부족하면 강화 판정·실패 보정·누적 시도 횟수가 변하지 않는가
- 수동 특수 강화에서 선택 재료가 0개면 시도되지 않는가
- 자동 단조에서 지정 재료가 0개면 해당 슬롯 없이 진행하는가
- 수동과 자동의 동일 단계·동일 선택 시 비용 계산이 같은가

## 자동 단조 확인

1. 목표 강화 단계를 지정한다.
2. 반복 여부와 단조 방식을 지정한다.
3. 특수 강화 보조재료·촉매를 지정한다.
4. 자동 단조를 시작한다.

확인 항목:

- 목표 단계까지 빠르게 진행하는가
- 특수 강화에서만 지정 재료를 소비하는가
- 지정 재료 재고가 없으면 해당 재료 없이 계속하는가
- 목표 도달 시 자동 보관하는가
- 반복 설정 시 새 철검으로 계속하는가
- 보관함이 가득 차면 중지하는가
- 골드 부족과 수동 중지에서 종료하는가
- 파괴 시 반복 설정에 따라 새 무기로 재시작하거나 종료하는가
- 장시간 실행 중 멈춤·무한 반복·심한 프레임 저하가 없는가

## 전체 제작 흐름

`scenes/main/main.tscn`을 현재 Scene으로 열고 F6를 누르면 다음 흐름을 확인할 수 있다.

```text
철검 제작 → 광클·피버 → 정밀 마감 → 일반/특수 강화 → 보관
```

## 키보드·터치

- Space: 현재 가능한 망치질·일반 강화·특수 강화 타격 실행
- 마우스 클릭과 모바일 터치도 동일 경로 사용

## 수동 검증 상태

다음은 자동 테스트로 대체하지 않는다.

- 실제 Godot 화면의 세로 스크롤·버튼 가독성
- +100 수동 완주 피로도
- 자동 단조 장시간 체감과 밸런스
- Android 실제 기기 터치·노치·안전 영역
- AAB·접근성·성능 프로파일

실행하지 않은 항목은 `NOT_RUN` 또는 `UNVERIFIED`로 기록한다.
