# Godot 강화 테스트 실행

## 필요한 버전

- Godot 4.7.1 stable
- 프로젝트 브랜치: `agent/blacksmith-mobile-foundation`

## 강화만 바로 테스트

1. Godot에서 저장소 루트의 `project.godot`을 연다.
2. FileSystem에서 `scenes/test/enhancement_test.tscn`을 연다.
3. `F6` 또는 우측 상단의 **현재 장면 실행**을 누른다.
4. 테스트용 철검 +0 상태에서 +20까지 강화한다.

## 강화 조작 규칙

- +1~+4: 버튼 한 번으로 즉시 강화 판정
- +5: 정밀 강화 후 첫 수식어 추가
- +6~+9: 버튼 한 번으로 즉시 강화 판정
- +10: 정밀 강화 후 첫 수식어 2티어 강화
- +11~+14: 버튼 한 번으로 즉시 강화 판정
- +15: 정밀 강화 후 두 번째 수식어 추가
- +16~+19: 버튼 한 번으로 즉시 강화 판정
- +20: 정밀 강화 후 두 번째 수식어 2티어 강화

보조재료는 각 5단계 구간에 누적된다. +5와 +15의 수식어 추가 시 해당 구간에서 가장 강하게 누적된 성질을 사용한다.

## 전체 제작 흐름 테스트

`scenes/main/main.tscn`을 열고 `F6`을 누르면 다음 흐름을 확인할 수 있다.

`철검 제작 → 광클·피버 → 정밀 마감 → 강화 진입 → +20 강화`

## 키보드

- Space: 현재 가능한 망치질·강화·정밀 타격 실행
- 마우스 클릭과 모바일 터치도 동일하게 지원

## 확인 항목

- 일반 강화 버튼을 눌렀을 때 타이밍 화면 없이 즉시 성공·실패가 표시되는가
- +5·+10·+15·+20에서만 정밀 강화 화면이 나타나는가
- 실패해도 단계가 내려가거나 무기가 파괴되지 않는가
- +5에서 첫 수식어, +10에서 첫 수식어 강화가 표시되는가
- +15에서 두 번째 수식어, +20에서 두 번째 수식어 강화가 표시되는가
- 720×1280 세로 화면에서 스크롤과 버튼이 정상 동작하는가
