# Godot 강화 테스트 실행

## 필요한 버전

- Godot 4.7.1 stable
- 프로젝트 브랜치: `agent/blacksmith-mobile-foundation`

## 강화만 바로 테스트

1. Godot에서 저장소 루트의 `project.godot`을 연다.
2. FileSystem에서 `scenes/test/enhancement_test.tscn`을 연다.
3. `F6` 또는 우측 상단의 **현재 장면 실행**을 누른다.
4. 테스트용 철검 +0 상태에서 +100까지 강화한다.

## 강화 조작 규칙

- 최대 강화 단계: +100
- 일반 단계: 버튼 한 번으로 즉시 강화 판정
- +5 단위: 정밀 강화 필수
- +10 단위: 보조재료·촉매 선택 영역 표시 + 정밀 강화

반복 예시:

```text
+1~+4 원클릭
→ +5 정밀 강화
→ +6~+9 원클릭
→ +10 보조재료·촉매 선택 + 정밀 강화
→ 같은 구조를 +100까지 반복
```

보조재료와 촉매는 +10·+20·+30·…·+100을 시도할 때만 화면에 나타나며 해당 시도에만 적용된다. +5·+15·+25 같은 중간 정밀 강화에서는 재료 선택 영역이 표시되지 않는다.

## 수식어 성장 이정표

- +10: 첫 수식어 1티어 추가
- +20: 첫 수식어 2티어
- +30: 두 번째 수식어 1티어 추가
- +40: 두 번째 수식어 2티어
- +50: 세 번째 수식어 1티어 추가
- +60: 세 번째 수식어 2티어
- +70: 첫 수식어 3티어
- +80: 두 번째 수식어 3티어
- +90: 세 번째 수식어 3티어
- +100: 모든 수식어 4티어

## 전체 제작 흐름 테스트

`scenes/main/main.tscn`을 열고 `F6`을 누르면 다음 흐름을 확인할 수 있다.

```text
철검 제작 → 광클·피버 → 정밀 마감 → 강화 진입 → +100 강화
```

## 키보드

- Space: 현재 가능한 망치질·강화·정밀 타격 실행
- 마우스 클릭과 모바일 터치도 동일하게 지원

## 확인 항목

- 일반 강화 버튼을 눌렀을 때 타이밍 화면 없이 즉시 성공·실패가 표시되는가
- +5 단위에서만 정밀 강화 화면이 나타나는가
- +10 단위에서만 보조재료·촉매 선택 영역이 나타나는가
- +5·+15·+25에서는 재료 선택 영역이 숨겨지는가
- 실패해도 단계가 내려가거나 무기가 파괴되지 않는가
- +10부터 수식어가 추가·강화되는가
- +100에서 강화가 완료되고 세 수식어가 4티어가 되는가
- 720×1280 세로 화면에서 스크롤과 버튼이 정상 동작하는가
