# MVP-001 제작 터치 수직 프로토타입

## 목표

Android 세로 화면에서 철검 제작의 첫 손맛을 검증한다.

## 플레이 흐름

`철 고정 → 제작 시작 → 광클·자동 작업 → 피버 → 선택적 정밀 마감 → 철검 완성 → 다시 제작`

## 포함

- 철 주재료와 검 무기 종류의 고정된 첫 조합
- 터치당 제작 진행도
- 기본 자동 작업
- 연속 터치로 차오르는 피버 게이지
- 피버 중 터치·자동 작업 배율
- ON/OFF 가능한 정밀 마감
- 움직이는 마감 포인터와 완벽·좋음·보통 판정
- 완성 결과 요약과 다시 제작
- 터치·마우스·Space 입력
- 상태 계산을 UI와 분리한 테스트 가능한 모델

## 제외

- 강화와 +5 수식어
- 고객·상인 판매
- 저장·방치 복귀
- 사운드·진동·완성 아트
- Android export와 실기기 검증

## 완료 기준

- 정밀작업 OFF로도 철검을 완성할 수 있다.
- 빠르게 연속 터치하면 피버가 발동하고 작업량이 증가한다.
- 정밀작업 ON에서는 타이밍에 따라 마감 품질이 달라진다.
- 한 번 완성 후 상태를 초기화해 다시 제작할 수 있다.
- 모델 단위 검증과 데이터 정적 검증이 통과한다.

## 현재 검증

- JSON 데이터 검증: PASS
- Godot 4.7.1 헤드리스 프로젝트 파싱: PASS
- 제작 모델 테스트 4건: PASS
- 실제 화면 시각 검수: NOT_RUN
- Android 실기기 터치 검증: NOT_RUN
