## 확정된 핵심 경험

1. 일반 제작은 빠른 연속 터치로 진행하고 과도한 광클은 피버타임을 발동한다.
2. 피버타임은 작업 속도와 제작·강화 성공 확률을 높인다.
3. 마무리와 강화에서는 선택적으로 정밀작업 타이밍 게임을 켤 수 있다.
4. 주재료로 철검·은검·운석검 같은 수식어 없는 기본 완성품을 만든다.
5. 강화 이정표에서 수식어가 추가·강화된다.
6. 주재료·보조재료·촉매 조합으로 강화 확률을 조절하고 숨은 무기를 발견한다.
7. 고객 직접 방문과 상인 납품 의뢰는 서로 다른 판매 판단을 제공한다.
8. 검투사·군대·모험가는 각각 명성, 안정 수익·생존, 고급 재료를 담당한다.

## 현재 구현

### MVP-001 제작

- 철검 제작 한 사이클
- 터치·자동 작업 진행
- 연속 터치 피버와 작업 배율
- ON/OFF 가능한 정밀 마감
- 완벽·좋음·보통 마감 결과
- 완성 철검을 강화 화면으로 전달하거나 다시 제작

### MVP-002 강화

- 철검 +0~+5 강화
- 숫돌·화염석·정령의 심장 보조재료 선택
- 살라맨더의 핵 촉매와 성공률 가산
- 정밀 강화 ON/OFF와 GOOD·PERFECT 가산
- 실패 시 단계 유지·파괴 없음·다음 성공률 보정
- 사용한 재료 성질 누적
- +5에서 `날카로운`·`불타는`·`정령이 깃든` 첫 수식어 획득
- +5 완료 후 새 철검 제작 복귀

### 검증

- JSON 데이터 검증 PASS
- Godot 4.7.1 프로젝트·Main Scene 파싱 PASS
- 제작 모델 4건 PASS
- 강화 모델 5건 PASS
