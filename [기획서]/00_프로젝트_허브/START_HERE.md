## 확정된 핵심 경험

1. 일반 제작은 빠른 연속 터치로 진행하고 광클은 피버 보상을 만든다.
2. 마무리는 선택적 정밀 타이밍으로 품질 차이를 만든다.
3. 일반 강화는 +10 단위를 제외한 단계에서 원클릭으로 진행한다.
4. 특수 강화는 +10·+20·…·+100에서 보조재료·촉매·정밀 판정과 수식어 성장을 제공한다.
5. 고단계일수록 공격력·판매가·비용·하락·파괴가 함께 가속된다.
6. 균형·안정·폭주 단조로 위험과 보상을 선택한다.
7. 폭주 단조는 낮은 확률로 총 2단계 상승하지만 특수 강화와 +9 끝자리에서 사용할 수 없다.
8. 강화한 무기는 최대 6칸 보관함에 저장한다.
9. 자동 단조는 목표 단계 도달 후 자동 보관하며 반복 시 보관함이 찰 때까지 진행한다.
10. 자동 특수 강화 재료가 없으면 해당 재료 없이 진행하고 이전 선택 효과를 누출하지 않는다.
11. 고객 직접 방문과 상인 납품은 서로 다른 판매 판단을 제공한다.

## 현재 구현

### 제작

- 철검 제작 한 사이클
- 터치·자동 작업·연속 터치 피버
- 선택적 정밀 마감과 품질 결과
- 완성 철검을 강화로 전달

### +100 강화·보관

- 일반 강화 / +10 단위 특수 강화 분리
- 공격력·판매가·다음 효과·비용·위험 표시
- 수식어 3개, +100에서 전체 4티어
- +11 단계 하락, +30 파괴 가능
- 실패 보정과 안정 단조 보호
- 목표 도달 전 수동 보관, 최대 6개 상세 확인

### 자동 단조

- 목표 단계·반복·단조 방식 지정
- 특수 강화 보조재료·촉매 지정
- 골드·재료 소비와 재료 부족 fallback
- 목표 도달 자동 보관
- 보관함이 찰 때까지 반복
- 골드 부족·보관함 가득 참·수동 중지·파괴 처리

### 자동 검증 이력

- JSON 데이터 검증 PASS
- Godot 4.7.1 프로젝트 파싱 PASS
- 강화 Scene 스모크 PASS
- 제작·강화 모델 테스트 PASS

실제 Android·시각·접근성·성능 검증은 별도이며 현재 `NOT_RUN`이다.
