# Active Context

## 현재 목표

Godot에서 제작→+100 강화→보관·자동 반복까지의 POC가 실제 화면과 Android 터치에서 이해 가능하고 재미있는지 검증한 뒤 첫 판매 루프로 연결한다.

## 현재 상태

- 제품 단계: Prototype
- 작업 게이트: Verification
- 버전 표시: `POC v0.6.0`
- 기본 실행: `project.godot` 열기 → `F5`
- 제작: 터치·자동 작업·피버·선택적 정밀 마감
- 강화: 일반/특수, +100, 성장·가격·비용·위험 미리보기
- 실패: +10까지 유지, +11부터 하락, +30부터 파괴
- 실패 보정: 실패당 +4%p, 최대 +24%p
- 단조 기술: 균형·안정·폭주
- 폭주: 8% 확률 총 2단계 상승, 특수 강화와 건너뛰기 위치 사용 불가
- 보관함: 메모리 기반 6칸
- 자동 단조: 목표 단계, 자동 보관, 가득 찰 때까지 반복, 골드·재료 소비
- 자동 검증: Godot 파싱·Scene smoke·제작 4건·강화 12건·JSON
- Android 실기기: NOT_RUN
- 저장·복귀: 미구현
- 고객·상인 판매: 미구현
