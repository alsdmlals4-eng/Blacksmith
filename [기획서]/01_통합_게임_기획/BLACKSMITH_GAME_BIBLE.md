# Blacksmith 통합 게임 기획서

## 1. 프로젝트 약속

플레이어는 모바일 화면을 빠르게 두드려 무기를 벼리고, 빠른 강화와 10단계 이정표의 특수 강화에서 공격력·수식어·가격을 키운 뒤, 위험을 감수해 만든 무기를 상황에 맞는 고객 또는 상인에게 판매해 대장간을 성장시킨다.

## 2. 핵심 재미

- 터치 직후 진행도·피버·타격 반응
- 9회의 빠른 일반 강화와 1회의 선택 중심 특수 강화
- 현재 공격력이 클수록 다음 성공 보상이 커지는 성장
- 성공·하락·파괴 확률과 가격을 보고 안전/도약을 선택
- 폭주 단조의 낮은 확률 2단계 도약
- 원하는 단계에서 멈추고 보관·판매하는 판단
- 자동 단조로 반복 생산하되 자원·위험을 그대로 부담

Base의 DDD는 의학적 진단이 아니라 첫 의미 있는 보상, 행동-피드백 지연, 보상 명료성·밀도, Micro→Session→Meta 보상 사다리를 점검하는 설계축으로만 사용한다.

## 3. 핵심 루프

```text
재료 획득
→ 기본 무기 제작
→ 일반·특수 강화
→ 보관
→ 고객 방문 또는 상인 납품
→ 골드·명성·고급 재료
→ 대장간 성장
```

현재 POC는 제작→강화→보관·자동 단조까지 구현됐고 판매·저장 경제는 다음 범위다.

## 4. 제작

### 4.1 일반 작업

- 모루·무기 터치로 진행도 증가
- 자동 작업으로 방치 진행
- 빠른 연속 터치로 피버 충전
- 직원 시스템 없음

### 4.2 피버

- 터치 작업량 증가
- 자동 작업 속도 증가
- 마무리 보너스
- 장기적으로 강화 보너스 연결 후보

광클에 과열·피로 불이익을 두지 않는다. 장시간 탭을 강제하지 않고 자동 진행을 병행한다.

### 4.3 정밀 마감

- 제작 마감에서 사용자가 ON/OFF
- ON이면 타이밍으로 품질 보너스
- OFF여도 제작 완료 가능
- 완벽·좋음·보통 결과

## 5. 기본 무기

- 주재료와 무기 종류로 이름·기본 성능 결정
- 기본 제작에는 수식어 없음
- 철+검 → 철검
- 은+검 → 은검
- 운석철+검 → 운석검

현재 구현은 철검 한 종류다.

## 6. 강화

### 6.1 일반 강화

- 목표 단계가 +10 단위가 아닐 때
- 버튼 한 번으로 즉시 판정
- 보조재료·촉매·정밀 판정 없음
- 이전 특수 강화의 선택 효과가 적용되지 않음

### 6.2 특수 강화

- +10·+20·…·+100
- 보조재료·촉매 선택
- 정밀 판정
- 수식어 추가·성장
- 높은 비용·성장·위험

### 6.3 성장과 가치

- 현재 강화 공격력 기준 비율 성장
- 일반 성장률은 낮고 특수 강화 성장률은 높음
- 10단계 구간이 높을수록 성장률 증가
- 비용과 판매가는 단계·공격력에 따라 가속
- 시도 전 현재/다음 공격력, 판매가, 비용, 성공·유지·하락·파괴 확률 표시
- 실제 수치 정본은 `data/crafting/enhancement_balance.json`

### 6.4 성공률과 실패 보정

- 10단계 성공률 패턴 반복
- 높은 구간일수록 기본 성공률 감소
- 실패당 +4%p, 최대 +24%p
- 성공 시 보정 초기화
- 하락 후에도 보정 유지

### 6.5 위험

- +1~+10 실패: 단계 유지
- +11부터 단계 하락
- +30부터 무기 파괴
- 고단계일수록 하락 비중·파괴 비중·하락 단계 증가
- 파괴 무기는 보관 불가
- 무기 수리 시스템 없음

## 7. 단조 기술

### 균형 단조

기본 확률·비용·성장·위험.

### 안정 단조

- 비용 1.8배
- 성장 0.95배
- 파괴 방지
- 하락 위험 절반

### 폭주 단조

- 성공 시 8% 확률로 총 2단계 상승
- 특수 강화에서 사용 불가
- 도약 범위에 +10 단위가 포함되면 사용 불가
- 자동 단조 목표를 넘길 수 있으면 균형 단조로 대체

## 8. 재료와 촉매

### 주재료

기본 무기 계열·성능을 결정.

### 보조재료

- 숫돌 → 날카로운
- 화염석 → 불타는
- 정령의 심장 → 정령이 깃든

특수 강화 수식어 방향에 영향을 준다.

### 촉매

- 살라맨더의 핵: 성공·성장·가치
- 수호 가루: 파괴 방지·하락 감소
- 폭주의 불씨: 성장·가치와 위험 동시 증가

가격·효과 정본은 `data/crafting/materials.json`.

## 9. 수식어

- +10 첫 수식어 1티어
- +20 첫 수식어 2티어
- +30 두 번째 수식어 1티어
- +40 두 번째 수식어 2티어
- +50 세 번째 수식어 1티어
- +60 세 번째 수식어 2티어
- +70·+80·+90 각 수식어 3티어
- +100 모든 수식어 4티어

새 슬롯은 이미 가진 수식어와 다른 후보를 우선한다.

## 10. 숨은 무기

특정 주재료·보조재료·촉매 조합이 고유 무기를 만든다.

예: 화염 계열 검 + 정령의 심장 + 살라맨더의 핵 → 살라맨더 화염검

- 발견 전 힌트
- 최초 발견 후 도감
- 틀린 조합은 가능하면 일반 결과
- 현재 변환 로직은 미구현

## 11. 보관함·자동 단조

### 보관함

- POC 용량 6
- 무기 이름·강화 단계·공격력·판매가·누적 비용·예상 손익·수식어·촉매 기록 표시
- 파괴 무기 보관 불가
- 현재 메모리 전용

### 자동 단조

- 목표 단계·단조 기술·보조재료·촉매 지정
- 목표 도달 시 자동 보관
- 보관함이 찰 때까지 반복
- 골드·재료 재고 소비
- 선택 재료가 없으면 빈 슬롯으로 계속
- 골드 부족·중지·가득 참에서 종료
- 파괴 시 반복 설정이면 새 철검으로 재시작

저장·복귀와 실제 상점 경제 연결은 후속이다.

## 12. 판매

### 고객 직접 방문

- 상황과 목적을 제시
- 기본 대금 고정
- 플레이어가 보관 무기 선택
- 적합도와 활동 결과로 나중에 인센티브
- 명성·단골·추가 보상

### 상인 납품

- 원하는 무기 종류·조건 지정
- 강화·수식어·희귀도 가산
- 납품 즉시 정산
- 재료·촉매·힌트 판매

두 방식은 `결과형 관계 보상`과 `즉시 조건형 경제 보상`으로 구분한다.

## 13. 고객

### 검투사

명성·인기·경기 인센티브·단골. 다음 MVP.

### 군대

안정 골드와 방어 전투력. 시즌 실패 시 영구 대장간 성장은 보존.

### 모험가

지역 요구에 맞는 무기로 희귀 재료 획득.

베팅은 확정 출시 범위가 아니며 고객 결과 루프 검증 뒤 재검토한다.

## 14. 모바일 UX·접근성

- 720×1280 세로형, Expand
- 한 손 플레이와 엄지 도달
- 충분한 터치 크기·간격
- 안전 영역·시스템 제스처
- 긴 화면 세로 스크롤
- 위험을 색만이 아니라 텍스트·수치로 표시
- 정밀 판정은 짧고 명확하며 자동 단조 대안 제공
- 진동·흔들림·모션 감소 옵션은 출시 범위에서 필요
- 연타를 강제하지 않고 자동 작업·자동 단조 제공
- 실제 기기 접근성 검증은 NOT_RUN
